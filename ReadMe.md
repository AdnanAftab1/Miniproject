# How to run 

 1. Copy the repository in your local system.
 2. Install ollama with a "gemini 3b" and "mxbai-embed-large"
 3. Open the project folder and run :
    
    Create an environement:

        python3 -m venv .venv
        python -m venv .venv

         
    Install dependencies:

        pip install -r req.txt

    Wait for a while ,then:

        streamlit run Chatbot.py
